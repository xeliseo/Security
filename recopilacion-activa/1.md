# 1. Transferencia de Zona
Una capacidad que tienen los nameservers (servidores DNS), una opción que no proporcionan para copiar todo el fichero de zona a un servidor DNS secundario de manera que no tengamos que crearlo desde 0.

En muchas ocaciones este servidor DNS está malconfigurado y nos permite realizar una transferencia de zona con un servidor DNS que no esté autorizado. Es decir, nosotros podríamos iniciar esa acción de transferencia de zona desde nuestro propio equipo, haciendonos pasar por un servidor DNS y si está mal configurado el servidor principal  lo que va a pasar es que nos va a enviar toda esa información que se encuentra en el fichero de zona.

## Visualizar registros DNS
- [DNSTransfer.com](https://dnstransfer.com)

## Comandos para transferencia de registros

Obtener ficheros de zona para un nameserver

```
nslookup
> set type=<type_record>
> <domain>

dig axfr <domain> @<dns>


dnsrecon -d <domain> -t axfr
```

**dnsrecon**
- Encuentra los nameservers
- Empieza a tratar de obtener el fichero de zona para cada uno de los nameservers encontrados
